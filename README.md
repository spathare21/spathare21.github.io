## ✨ Features

- **Dark theme** with teal accent colors
- **Fully responsive** — mobile-first layout
- **Animated timeline** for work experience
- **Scroll reveal** animations via Intersection Observer
- **Typed text effect** in hero section
- **No dependencies** — pure HTML, CSS, and vanilla JS

## 🚀 Deploy to GitHub Pages

1. Push this repository to GitHub as `spathare21.github.io`
2. Go to **Settings → Pages**
3. Under **Source**, select `main` branch and `/ (root)` folder
4. Click **Save** — your site will be live at `https://spathare21.github.io`
